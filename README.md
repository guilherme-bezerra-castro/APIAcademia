# API REST em .NET
API REST desenvolvida em C# com .NET e Entity Framework Core, integrada ao MySQL, responsável por operaçőes CRUD de usuários.

## Tecnologias
- C#
- .NET
- Entity Framework Core
- MySQL

## Funcionalidades
- CRUD de usuários

## Estrutura
- Controllers
- Models
- Data

## Como executar
1. Clone o repositório
2. Configure o banco no appsettings.json
3. Rode: dotnet ef database update
4. Execute: dotnet run

- API REST
